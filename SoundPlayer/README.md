# SoundPlayer

Sound visualizer width Picture
coded by Openframeworks

## Usage
 読み込んだ夜景写真の光（白点）に重ねて円を描く。
 読み込んだ音楽ファイルに応じて光（白点）の大きさを変化させるプログラム。
OFがインストールされたマシンでbin/SoundPlayerDebugをダブルクリックしてください。
